# Para C Language
---

## Brief
C-style language 

## Using
Flex - A tool for generating lexical analyzers (scanners) from regular expressions.
Bison - A parser generator that builds syntax analyzers from context-free grammars.

## How to build
```shell
    cmake -DCMAKE_BUILD_TYPE=Release -S . -B build
    cmake --build build
    ./build/para_cl.x
```
